19 century Art Website

  • Outcome: A responsive website written in HTML5, CSS3, JavaScript, BootStrap, Flask, with a user system connected to the MySQL database.
  • Description: This project is aiming to develop and maintain a website which introduces the 19 century art history for the students in the School of Arts, Unversity of Michigan.
  • My role: Web Developer
  • Skills: HTML5, CSS3, JavaScript, Django
  • Link: The Website is still in the testing stage, therefore the code is not allowed to released to the public.

Map

The map tool is aiming to provide the mapping of 19 century theaters and palces where historial events in art history took place. The map could zoomed in and it could automatically generate the items one by one on the map.
