Life Expectancy Analysis

  • Outcome: A data analysis report regards the expectancy life length
  • Description: This Project is part of the Course Data Analysis and Statistics. This project is intend to find out the factors that effect the life expectancy by performing a linrear regression regarding factors from various aspects that might effect life expectancy.
  • My role: Data Analyst
  • Skills: Excel, R
  • Report: Download

Linear Regression Model

The Analysis process is consists of five sections:
1. Initializing regression: regarding 5 fields that could effect the life expectancy, we choose 8 factors to initialize the regression.
2. Adjust Regression: We examined the colliearity problem and used BP test to exclude the outliers of the regression.

Prediction


3. Making Prediction based on the regression: Based on the regression, we utilize the R to make prediction of the life expectency.
4. Using three countries to test the regression: To examine the regression, we use three countries to test the difference between estimated and true life expectency.

Hypothesis Testing

5. Running Hypothesis testing: We propose a hypothesis that the life expectancy between diffent continent has no significant difference and using t-statistics to test it.
