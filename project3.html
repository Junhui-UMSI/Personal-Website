World Cup 2014

Project Overview

  • This website project is part of the course Web Design and Develop.This website is intend to be a news website for the World Cup 2014 for students in Zhejiang University. Students could browse the news as well as the funny stories in the World Cup through the webstie. Besides, it provides a user system which allows students to log in and comment on the news.
  • My role: Web Developer&Designer | UX researcher
  • Method| Skills: HTML5, CSS 3, JavaScript, JQuery

    • Website View

    The Website is consist of four pages, the news, the statistics, the funny story and the user comment page.
